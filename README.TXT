PROJECT TITLE: SudokuSolver
PURPOSE OF PROJECT: This project provides a terminal based GUI to solve the classic Sudoku 9x9 puzzle.
                    There are two options for the user - to enter a puzzle on their own, or - to see a custom puzzle getting solved
                    Further this project provides the option of viewing the puzzle being solved by the computer using hit and trial method (with backtracking handling the logic)
HOW TO START THIS PROJECT:From Bluej IDE for Java 
                            1) Open the directory as a Bluej project
                            2) Compile all the classes using the compile button
                            3) Right click the Sudoku class and run its main method
                            (How to install Bluej: https://youtu.be/OHdZJZETPFQ)
                           
                           To run without BlueJ(Command-line)
                            1) Navigate to the project folder and compile the Sudoku class using "javac Sudoku.java" command(without quotes)
                            2) Run the program using "java Sudoku" command(without quotes)
